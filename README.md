# 智简魔方财务对接DNSLA二级域名模块

## 核心功能

  * **自动化供应**：通过魔方财务自动化接口创建二级域名进行解析
  * **客户端功能**：客户可在魔方财务的用户中心管理和随意解析二级域名。
  * **安全隔离**：插件代码全开源，没有任何后门。
